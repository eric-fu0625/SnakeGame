# Snake Game (Java Swing)

A classic Snake game developed using Java Swing

# Characteristics

Control the snake to eat the food and avoid colliding with itself.
Automatic speed increase (accelerates by 50 points every time)

# Operating Requirements

- Java 8 or higher version
- Java environment supports Java Swing

# How to run the game

-> javac SnakeGame.java
-> java SnakeGame
